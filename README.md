# 🚀 CodeLeap Network - Frontend Challenge

A modern, responsive CRUD application built for the CodeLeap recruitment process. This project simulates a social network where users can post, edit, and delete messages.

## 🛠 Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS
- **Components:** Shadcn/ui (Radix UI)
- **Icons:** Lucide React
- **API Consumption:** Native Fetch API with custom service layer

## ✨ Features & Bonuses Implemented

Beyond the basic requirements, I've added several professional features to enhance UX:

- [x] **Full CRUD:** Create, Read, Update, and Delete posts.
- [x] **Session Persistence:** User session is saved in `localStorage`. Refreshing the page won't log you out.
- [x] **Pagination (Load More):** Efficient data fetching using `limit` and `offset` to handle large feeds.
- [x] **UX Feedback:** - **Skeleton Loaders:** Beautiful ghost elements while fetching initial data.
  - **Button Spinners:** Real-time feedback during API mutations.
  - **Smooth Animations:** Fade-in and slide-up effects for new posts.
- [x] **Responsive Design:** Optimized for Mobile, Tablet, and Desktop.
- [x] **Professional UI:** Strict adherence to the provided design with Roboto font and custom color palette.
